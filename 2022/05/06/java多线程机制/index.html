三种方法的区别

1.继承Thread
优点：编写代码比较简单，可以直接使用Thread类中的方法
缺点：扩展性比较差，因为继承了一个类就不能再继承其他的类
2.实现Runnable和Callable接口
优点：扩展性比较好，在实现接口的同时还可以继承其他的类
缺点：编写代码比较复杂，而且不能直接使用Thread类中的方法
